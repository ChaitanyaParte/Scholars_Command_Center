# 🎓 Falcons — The Scholar’s Command Center

> A unified nervous system for a student’s academic life.

---

## 🚀 Overview

**Nexus-Uni** is a modern, frontend-focused academic dashboard designed to bring together everything a student needs into one powerful interface.

Instead of juggling multiple apps for notes, assignments, feedback, and resources, Nexus-Uni provides a **single, intelligent “cockpit”** that helps students:

* Stay organized
* Prioritize effectively
* Track academic health
* Access resources instantly
* Plan their future

---

## 🎯 Problem Statement

Students often face a **fragmented academic experience**, using different platforms for:

* Assignments
* Notes
* Feedback
* Resources
* Planning

This leads to confusion, inefficiency, and poor prioritization.

👉 **Nexus-Uni solves this by consolidating everything into one high-density, clean, and interactive dashboard.**

---

## ✨ Core Features

### 📊 1. Priority Hub (Task Gravity)

Tasks are automatically prioritized using:

> **Task Gravity = Urgency × Grade Weight**

* 🔴 High Priority
* 🟡 Medium Priority
* 🟢 Low Priority

Helps students focus on what truly matters.

---

### 💬 2. Feedback Loop (GitHub-style)

Professor feedback is treated like **issues**:

* View comments
* Reply to feedback
* Mark as resolved ✅

👉 Makes feedback actionable instead of ignored.

---

### 📚 3. Resource Shelf

A smart, searchable library for study materials:

* Upload and organize files
* Tag-based filtering
* Instant search 🔍
* 👀 Preview files inside app
* ⬇️ Download resources

---

### 🤖 4. Context-Aware UI (Adaptive Dashboard)

The interface dynamically adapts based on student activity.

Example:

* During class → shows **Live Notes + Resources only**
* After class → shows **Tasks + Feedback + Dashboard**

👉 Reduces distractions and improves focus.

---

## 🌟 Additional Features

Visual representation of how subjects/topics are connected.

### ❤️ Subject Health

Track academic performance using visual indicators:

### 🎯 College Predictor

Predict possible colleges based on performance:

* Input rank / score

* Helps in planning future goals

## 🧠 Design Philosophy

### ⚖️ Density vs Clarity

Display **rich information** without overwhelming the user.

* Card-based UI
* Visual hierarchy
* Minimal clutter

---

### ⚡ Interaction First

* Fast navigation
* Smooth UI transitions
* Keyboard-friendly design
